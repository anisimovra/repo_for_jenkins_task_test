Описание
========
Данные скрипты предназначены для автоматизации импорта изменений исходного кода эталонной схемы в master ветку репозитория git.
При первом запуске будет запрошен секретный пароль к ssh ключу. Пароль запомнится до перезагрузки системы.
Для запуска выполнить run.bat

Требования к окружению
======================
Должны быть установлены
1. CFT Platfrom IDE версии 2.24 и выше (версия от 21/05/2019 или позже).
2. Git последней версии https://git-scm.com/downloads
3. Oracle client версии 11.2.0.4 и выше.

Должны быть настроены:
1. Git в соответствии с документацией CFT Platform IDE Documentation > Настройка Git/GitLab/Bitbucket > Настройка Eclipse для работы с Git.
2. Настроен ssh в соответствии с той же документацией.
3. Настроен путь до tns: Window > Preferences > CFT Platform > Подключения

Настройка переменных
====================
Перед первым запуском скриптов необходимо изменить значения переменных в файле run.bat.
WORKING_DIR_PATH	Путь рабочего каталога
СOMMIT_TEXT 		Текст коммита для определения даты последней синхронизации.
ECLIPSE_PATH		Путь до папки с CFT Platfrom IDE
SCHEMA				Схема для получения патча. Путь до TNS должен быть прописан в А2
OWNER				Владелец схемы
USER				Пользователь
PASSWORD			Пароль пользователя
